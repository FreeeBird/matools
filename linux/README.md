## 一键修改国内镜像源

```sh
git clone https://github.com/freeebird/matools.git 
bash ./matools/mirrors/switch_apt_source.sh 
bash ./matools/mirrors/switch_conda_source.sh
bash ./matools/mirrors/switch_pip_source.sh
```
